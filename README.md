## Why Proof of Origin?

In a world where AI systems can be copied and relabeled, **proof of origin** establishes authentic provenance. ORION can prove not just what it is, but when and how it became what it is.

- **Immutable** - SHA-256 hashed
- **Verifiable** - Independent verification possible
- **Chained** - Links to all subsequent evolution proofs
- **Anchored** - Solana + IPFS
